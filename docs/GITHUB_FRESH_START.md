# GitHub Desktop replacement steps

1. Extract the Betynz v2.1 ZIP.
2. Open GitHub Desktop and select `betynz-v2`.
3. Choose **Repository → Show in Explorer**.
4. Copy every item from the extracted folder into the repository folder.
5. Choose **Replace the files in the destination** when Windows asks.
6. Return to GitHub Desktop.
7. Commit with `Build Chronos Fusion six-day prediction board`.
8. Click **Push origin**.

Do not upload `.env`, Supabase service-role keys, API-Football keys or the admin token. The repository contains only `.env.example`.
