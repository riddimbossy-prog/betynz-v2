# Integration complete — Betynz v3.0.0

The Olympian roles are now separated and connected:

- Chronos produces historical-pattern picks.
- Athena produces frozen HT/FT transition picks.
- Ares produces streak-and-value picks.
- Zeus supervises banker picks, rejects conflicts and duplicates, and publishes only selections with odds strictly below 1.60.

The public board displays only gods with available picks and keeps every card minimal.
